Bootstrap学习笔记(三)-排版基础


##bootstrap排版

  • ###响应式图片

    在 Bootstrap 版本 3 中，通过为图片添加 .img-responsive 类可以让图片支持响应式布局。其实质是为图片设置了 max-width: 100%;、 height: auto; 和 display: block; 属性，从而让图片在其父元素中更好的缩放。

  <center>
![mark](http://ok8z4bh5q.bkt.clouddn.com/blog/20170311/065040116.png)
</center>
<center>*网页缩小前*</center>
<center>
![mark](http://ok8z4bh5q.bkt.clouddn.com/blog/20170311/065129812.png)
</center>
<center>*网页缩小后*</center>

代码:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Dome08</title>
    <link rel="stylesheet" href="../node_modules/bootstrap/dist/css/bootstrap.min.css">
</head>
<body>
<div class="container">
    <div class="row">
        <div class="col-md-3"><img src="imangs/timg.jpg" class="img-responsive"/></div>
        <div class="col-md-3"><img src="imangs/23.jpg" class="img-responsive "/></div>

</div>

  • ###副标题small
    在标题内还可以包含 标签或赋予 .small 类的元素，可以用来标记副标题。

  • ###列表
    .list-unstyled可以将列表的圆点或数字样式去掉
    .list-inline可以将列表放在一行，便于做导航
    .dl-horizontal可去掉定义列表中说明的缩进并将定义词和说明放在一行、



代码：

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Dome02</title>
    <link rel="stylesheet" href="../node_modules/bootstrap/dist/css/bootstrap.min.css">
</head>
<body>
<div class="container">
        <ul class="list-unstyled"><!--去掉无序列表的圆点-->
            <li>中国</li>
            <li>美国</li>
            <li>日本</li>

        <hr>
        <ol class="list-unstyled"><!--去掉有序列表的数字-->

            <li>美国</li>
            <li>日本</li>
        </ol>

        </ol>
        <hr>

  • ###响应式表格
    将任何 .table 元素包裹在 .table-responsive 元素内，即可创建响应式表格，其会在小屏幕设备上（小于768px）水平滚动。当屏幕大于 768px 宽度时，水平滚动条消失。


代码：

<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Dome11</title>
    <link rel="stylesheet" href="../node_modules/bootstrap/dist/css/bootstrap.min.css">
</head>
