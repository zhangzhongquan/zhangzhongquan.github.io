Bootstrap学习笔记(二)


##bootstrap栅格统的使用方法

  • ###布局容器和列组合

    bootstrap 需要为页面内容和栅格系统包裹一个.container容器。我们提供了两个作此用处的类。注意，由于 padding 等属性的原因，这两种 容器类不能互相嵌套。

    1. .container 类用于固定宽度并支持响应式布局的容器。
    2. .container-fluid 类用于 100% 宽度，占据全部视口（viewport）的容器。

###例1：移动设备和桌面屏幕






代码:

<!DOCTYPE html>
<html lang="zh">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Dome02</title>
    <link rel="stylesheet" href="../node_modules/bootstrap/dist/css/bootstrap.min.css">
    <style>
        div{
            border:1px solid #333333;
        }
    </style>
</head>
<body>
<div class="container">
    <div class="row">
        <div class="col-xs-12 col-md-8">.col-xs-12 .col-md-8</div>
        <div class="col-xs-6 col-md-4">.col-xs-6 .col-md-4</div>
    </div>
    <div class="row">
        <div class="col-xs-6 col-md-4">.col-xs-6 .col-md-4</div>
        <div class="col-xs-6 col-md-4">.col-xs-6 .col-md-4</div>
        <div class="col-xs-6 col-md-4">.col-xs-6 .col-md-4</div>
    </div>
    <div class="row">
        <div class="col-xs-6 ">.col-xs-6 </div>
        <div class="col-xs-6 ">.col-xs-6 </div>
        <div class="col-xs-6 ">.col-xs-6 </div>

    </div>
</div>
<script src="../node_modules/jquery/dist/jquery.min.js"></script>
<script src="../node_modules/bootstrap/dist/js/bootstrap.min.js"></script>
</body>
</html>

  • ###列偏移：

    使用 .col-md-offset- 类可以将列向右侧偏移。这些类实际是通过使用 选择器为当前元素增加了左侧的边距（margin）。例如，.col-md-offset-4 类将 .col-md-4 元素向右侧偏移了4个列（column）的宽度。

例：列偏移

<center>![mark](http://ok8z4bh5q.bkt.clouddn.com/blog/20170308/045132193.png)</center>
<center>***偏移前***</center>

<center>![mark](http://ok8z4bh5q.bkt.clouddn.com/blog/20170308/045856303.png)</center>
<center>***偏移*后**</center>
代码：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Dome02</title>
    <link rel="stylesheet" href="../node_modules/bootstrap/dist/css/bootstrap.min.css">
    <style>
        div{
            border: 1px solid #000000;
        }
    </style>
</head>
<body>
<div class="container">
    <div class="row">
        <div class="col-md-4">.col-md-4</div>
        <div class="col-md-4">.col-md-4</div>
        <div class="col-md-4">.col-md-4</div>
    </div>
    <div class="row">
        <div class="col-xs-6">.col-xs-6</div>
        <div class="col-xs-6">.col-xs-6</div>
    </div>
    <div class="row"><!--列偏移开始-->
        <div class="col-md-3 col-md-offset-2">col-md-3</div>
        <div class="col-md-2 col-md-offset-3">col-md-2</div>
    </div><!--列偏移结束-->
</div>
<script src="../node_modules/jquery/dist/jquery.min.js"></script>
<script src="../node_modules/bootstrap/dist/js/bootstrap.min.js"></script>
</body>
</html>

  • ###列嵌套：
    为了使用内置的栅格系统将内容再次嵌套，可以通过添加一个新的 .row 元素和一系列 .col-sm- 元素到已经存在的 .col-sm- 元素内。被嵌套的行（row）所包含的列（column）的个数不能超过12（其实，没有要求你必须占满12列）。

    例：列嵌套


    嵌套前

    嵌套后

    代码：

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1">
<title>Dome05</title>
<link rel="stylesheet" href="../node_modules/bootstrap/dist/css/bootstrap.min.css">
<style>
    div{
        border: 1px solid #000000;
    }
</style>
</head>
<body>
<div class="container">
<div class="row">
    <div class="col-md-4">.col-md-4</div>
    <div class="col-md-4">.col-md-4</div>
    <div class="col-md-4">.col-md-4</div>
</div>
<div class="row">
    <div class="col-xs-6">.col-xs-6</div>
    <div class="col-xs-6">.col-xs-6</div>
</div>
<div class="row"><!--列偏移-->
    <div class="col-md-3 col-md-offset-2">col-md-3</div>
    <div class="col-md-2 col-md-offset-3">col-md-2</div>
</div>
<div class="row"><!--列嵌套开始-->
    <div class="col-xs-8">
        <div class="row">
            <div class="col-xs-6">6</div>
            <div class="col-xs-6">6</div>
        </div>
    </div>
    <div class="col-xs-4">4</div>
</div><!--列嵌套结束-->
</div>
<script src="../node_modules/jquery/dist/jquery.min.js"></script>
<script src="../node_modules/bootstrap/dist/js/bootstrap.min.js"></script>
</body>
</html>

  • ###列排序

    通过使用 .col-md-push- （向右推）和 .col-md-pull-（向左拉） 类就可以很容易的改变列（column）的顺序。

    例:列排序


    排序前


    排序后

    代码：

    <!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1">
<title>Dome06</title>
<link rel="stylesheet" href="../node_modules/bootstrap/dist/css/bootstrap.min.css">
<style>
    div{
        border: 1px solid #000000;
    }
</style>
</head>
<body>
<div class="container">
<div class="row">
    <div class="col-md-4">.col-md-4</div>
    <div class="col-md-4">.col-md-4</div>
    <div class="col-md-4">.col-md-4</div>
</div>
<div class="row">
    <div class="col-xs-6">.col-xs-6</div>
    <div class="col-xs-6">.col-xs-6</div>
</div>
<div class="row"><!--列偏移-->
    <div class="col-md-3 col-md-offset-2">col-md-3</div>
    <div class="col-md-2 col-md-offset-3">col-md-2</div>
</div>
<div class="row"><!--列嵌套开始-->
    <div class="col-xs-8">
        <div class="row">
            <div class="col-xs-6">6</div>
            <div class="col-xs-6">6</div>
        </div>
    </div>
    <div class="col-xs-4">4</div>
</div><!--列嵌套结束-->
<div class="row"><!--列排序开始-->
    <div class="col-md-3 col-md-push-7">col-md-3</div>
    <div class="col-md-2 col-md-pull-2">col-md-2</div>
</div><!--列排序结束-->
</div>
<script src="../node_modules/jquery/dist/jquery.min.js"></script>
<script src="../node_modules/bootstrap/dist/js/bootstrap.min.js"></script>
</body>
</html>

  • ###清除浮动
    在某些阈值时，某些列可能会出现比别的列高的情况。为了克服这一问题，建议联合使用 .clearfix

    例：清除浮动


    清除前超阈值宽屏

    清除浮动前超阈值窄屏


    清除浮动前超阈值窄屏

    ![mark](http://ok8z4bh5q.bkt.clouddn.com/blog/20170308/060139562.png)</center>

    清除浮动后超阈值窄屏

    代码：

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Dome07</title>
    <link rel="stylesheet" href="../node_modules/bootstrap/dist/css/bootstrap.min.css">
    <style>
        div{
            border: 1px solid #333333;
        }
    </style>
</head>
<body>
<div class="container">
    <div class="row">
        <div class="col-xs-6 col-sm-3">col-xs-6 col-sm-3  1<br/>afdkfaskdfsldfjlaskdfsfa</div>
        <div class="col-xs-6 col-sm-3">col-xs-6 col-sm-3  2</div>
        <div class="clearfix visible-xs"></div> <!--清除上一行的浮动-->
        <div class="col-xs-6 col-sm-3">col-xs-6 col-sm-3  3</div>
        <div class="col-xs-6 col-sm-3">col-xs-6 col-sm-3  4</div>
    </div>
</div>

</html>
